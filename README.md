## Installation

The project requires Vite to run.
Install Vite as dependencies...
```sh
npm install vite --save-dev
```

Launch project...
```sh
npm run dev
```

Verify the deployment by navigating to your server address in
your preferred browser.

```sh
> react-training@0.0.0 dev
> vite
```
